# HerCode 解释器

使用 PHP 实现的 HerCode 代码解释器。

## 用法

```
php interpreter.php demo.hc
```

## 语法

```
Hello! Her World

function 那咋啦:
    say "啊啊啊啊啊啊啊啊啊啊啊左转的红灯最难等了"
end

start:
    那咋啦
end
```
